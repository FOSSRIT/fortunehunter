To run MPG follow this instructions:

Using terminal change to the directory that you cloned the files to.
run the command "cp images /home/olpc/images"
run the command cat main.py
Select the text that cat produces and copy it to the clipboard
Open up the Pippy program and copy the text
Click Run

For controls and update game information please visit:
http://wiki.sugarlabs.org/go/MPG
